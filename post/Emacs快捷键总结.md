# Emacs快捷键总结

> M 代表 alt  
> C 代表 Ctrl

	C-x C-c		退出Emacs
	C-g		退出一个正在运行中的命令

	C-v		向下移动一屏
	M-v		向上移动一屏

	C-l		重绘屏幕，并将光标置于屏幕中央

	C-p		移动到上一行
	C-n		移动到下一行

	C-b		向左移动一个字符
	C-f		向右移动一个字符

	M-f，M-b	一个词一个词的跳

	C-a，C-e	移动到一行的头部和尾部
	M-a，M-e	移动到一句的头部和尾部

	Shift-M-<	移动到文章的开头
	Shift-M-> 	移动到文章的末尾

	C-u 8 C-f	向前移动8个字符
	C-u 4 C-b	向后移动4个字符
	C-u 6 C-n	向下移动6行
	C-u 5 C-p	向上移动5行

	C-x d		打开目录
	
	C-x 0		关闭当前窗口
	C-x 2		早现编
	C-x 3		在右边新建一个窗口
	C-x o		移动到另一个窗口
	C-M-v		移动到下一个窗口
